UI Components

WeatherScreen()
Root composable that manages state routing and displays appropriate screen based on WeatherState (Loading, Success, Error)
Root Composable
Screen Composables
LoadingScreen()
Displays a circular progress indicator while fetching weather data from the API
SuccessScreen()
Main dashboard displaying current weather, details grid, 7-day forecast, and search section
ErrorScreen()
Shows error message with retry button for network failures or invalid coordinates
Weather Cards
CurrentWeatherCard()
Displays large temperature, weather icon, condition description, and "feels like" temperature
DetailCard()
Reusable metric card showing title and value (humidity, wind speed, precipitation, etc.)
ForecastCard()
Individual day forecast with date, weather icon, high/low temps, and rain probability
Utility Components
WeatherDetailsGrid()
Grid layout containing DetailCard components for humidity, wind, precipitation, and wind direction
SearchCoordinatesSection()
Form with latitude/longitude input fields and search button for location updates
