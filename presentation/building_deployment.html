Building & Deployment

Prerequisites
Android Studio
Giraffe or newer
JDK
Version 17+
Android SDK
API 34 recommended
Gradle
Version 8.2.0
Quick Build
$ cd climo-android $ ./gradlew build $ ./gradlew installDebug
Run on Device
Option 1: Android Studio
Open project → Click Run (Shift+F10)
Option 2: Command Line
./gradlew installDebug
Release APK
$ ./gradlew assembleRelease
Output: app/build/outputs/apk/release/app-release.apk
Default Location
App launches with New York City weather by default. Use coordinate search to change location. Data fetched in real-time from Open-Meteo API.
